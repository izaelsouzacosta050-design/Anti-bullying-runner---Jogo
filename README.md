# Anti-Bullying Runner

Endless Runner educativo, offline e sem dependências externas.

## Arquivos

- `index.html`: interface, telas e canvas.
- `style.css`: layout responsivo, temas e animações.
- `script.js`: `GameEngine`, gameplay, áudio, loja, dilemas e persistência.

## Como executar

1. Abra `index.html` em um navegador moderno.
2. Permita áudio ao tocar em qualquer botão.
3. Use teclado ou controles de tela para jogar.

## Controles

- `←` / `A`: mover para a esquerda.
- `→` / `D`: mover para a direita.
- `↑` / `W` / `Espaço`: pular.
- `↓` / `S`: deslizar/abaixar.
- Botões na tela: movimento no mobile.

## Recursos

- Canvas 2D com cenário roxo e pistas cyan.
- Tela de carregamento animada por 3,5 s.
- 3 pistas, trens, moedas e gemas.
- Loja com skates, bikes, ímã e escudo.
- 7 dilemas antibullying com XP positivo/negativo.
- XP, níveis 1 a 50 e certificado de Embaixador da Paz.
- LocalStorage para nome, avatar, XP, nível, moedas, gemas e recordes.

## Observações

- O jogo funciona offline.
- O áudio usa Web Audio API e só inicia após interação do usuário.
